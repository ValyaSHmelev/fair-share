# FairShare

Веб-приложение для деления общего счёта между друзьями после тусовок и выездов на природу.
Создаёте мероприятие, добавляете участников и позиции расходов, отмечаете, на кого делить каждую позицию — приложение считает, кто сколько должен скинуть.

Подробное ТЗ: см. [`SPEC.md`](./SPEC.md).

## Возможности

- Мероприятия с выбором валюты и даты
- Участники (тёзки допустимы) и группы/пары с суммарным итогом
- Позиции расходов с делением **поровну** между выбранными участниками
- Округление долей до целого рубля без расхождений (метод наибольших остатков)
- Отчёт в формате Excel-таблицы + экспорт в CSV
- Хранение в `localStorage`, экспорт/импорт всех данных в JSON (режимы «Заменить»/«Объединить»)
- Тёмная и светлая темы, адаптивный интерфейс

## Технологии

Vite • Vue 3 (`<script setup>`, TS) • Pinia • Vue Router • PrimeVue (тема Aura) • Vitest

## Запуск

```bash
npm install      # установка зависимостей
npm run dev      # запуск дев-сервера (http://localhost:5173)
npm run build    # продакшн-сборка (vue-tsc + vite)
npm run preview  # предпросмотр сборки
npm run test     # unit-тесты расчётов
```

## Логика расчёта

Каждая позиция делится поровну между отмеченными участниками. Доли округляются до целого
рубля так, чтобы их сумма точно равнялась округлённой цене позиции. Итог по участнику —
сумма его долей по всем позициям. Группы лишь агрегируют итоги своих участников и не влияют
на сам расчёт.
